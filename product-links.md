# Product Links for video-equipment.html

## Instructions for Cursor:

In assets/style.css add these new rules:

```css
.product-links {
  display: flex;
  gap: 6px;
  align-items: center;
  margin-top: 4px;
  font-size: 13px;
}
.product-links a {
  color: var(--link);
  text-decoration: none;
}
.product-links a:hover {
  text-decoration: underline;
}
```

In assets/style.css change .product-row to:
```css
.product-row {
  display: flex;
  flex-direction: column;
  align-items: flex-start;
  gap: 4px;
}
```

In pages/video-equipment.html, add a `<div class="product-links">` after each product's `.product-desc` div with the links below. Do NOT add links for: "Dein Smartphone", "Tageslicht", "Weiterhin dein Smartphone".

Format:
```html
<div class="product-links">
  <a href="AMAZON_URL" target="_blank">Amazon</a> · 
  <a href="IDEALO_URL" target="_blank">Idealo</a>
</div>
```
